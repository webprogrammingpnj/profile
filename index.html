Tentang Mata Kuliah

ditulis oleh irwanprasetya. pada 15 Maret 2024

Mata kuliah ini membahas konsep-konsep dasar, bagaimana merancang website menggunakan HTML, CSS, PHP dan pengolahan data menggunakan database Mysql. Serta bagaimana cara membuat domain website dan hosting website.

Capaian Pembelajaran Lulusan Program Studi Yang Dibebankan Pada Mata Kuliah (CPL) : Mampu merancang Website untuk PRODUK pada Mata Kuliah KEWIRAUSAHAAN dan mampu melakukan pengolahan data menggunakan database.
